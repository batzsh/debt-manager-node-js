# Debt Manager

Account all of your debts

## How to run

1. `yarn install` to install all packages
2. `yarn typeorm migration:run` to run all migrations
3. `yarn dev` to start dev server
4. Use [the collection](https://github.com/henbalmant/debt-manager/blob/master/debt-manager-collection.json) to test endpoints
